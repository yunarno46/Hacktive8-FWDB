Tentang Pemrograman

Apakah kamu pernah memiliki latar belakang pemrograman?

Saya adalah salah satu mahasiswa semester akhir di Universitas Bina Nusantara, Jurusan Teknik Informatika. Saya telah mempelajari beberapa bahasa pemrograman dasar, misalnya C, C++, C#, dan Java. Namun untuk pemrograman berbasis web, saya baru mempelajari satu tahun terakhir ini.

Kenapa kamu ingin belajar pemrograman?

Bagi saya, belajar pemrograman merupakan hal yang menarik. Kita dilatih untuk berfikir dengan algoritma yang terstruktur ketika menghadapi suatu kasus dalam coding. Tentu saja menurut saya ini sangat bermanfaat juga dalam dunia nyata dalam mencari problem solving.

Apa tujuan kamu setelah menyelesaikan program ini?

Harapan saya setelah menyelesaikan program Hacktive8 ini yaitu saya mampu membuat website profesional dengan tampilan yang menarik dari sisi UI dan fungsional dari sisi UX.

Sebutkan hobi dan interestmu!

Saya memiliki hobi musik dan game mobile maupun PC. Karena hobi inilah saya memiliki interest dengan dunia pemrograman, khususnya koding.
